# Arch + Niri dotfiles management

## 🚧 Work in progress

These are my personal dotfiles for Arch and the Niri window manager.

* **minimal**, easy to understand
* **non-LLM** generated
* absolutely **not perfect** and only meant to be a **reference**

## ✨ Highlights

### Waybar customization

![Waybar Customization](Images/WaybarScreenshot.png)

You can find the waybar files here:

* [config.jsonc](.config/waybar/config.jsonc)
* [style.css](.config/waybar/style.css)
* [mountains.css](.config/waybar/colors/custom/mountains.css) (Rose Pine palette)

## ⚙️Getting started

I manage these using a **bare repository**.

If you're unfamiliar with this method, I recommend this guide:
[DistroTube: Manage Dotfiles with a Bare Git Repo](https://www.youtube.com/watch?v=tBoLDpTWVOM)
